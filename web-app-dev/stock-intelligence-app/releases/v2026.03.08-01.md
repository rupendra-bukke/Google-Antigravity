## Summary

- Timeline cards now show clear date/time context for each stored response.
- Dev/Prod app URLs are documented in workflow reference docs.

## Changes

### Added
- EOD reconcile scheduler jobs:
  - `15:31 IST` primary reconcile
  - `15:36 IST` retry reconcile
- Manual reconcile API:
  - `POST /api/v1/checkpoints/reconcile?date=YYYY-MM-DD`
- Timeline date/time visibility:
  - board data date in header
  - captured timestamp in each populated card

### Changed
- Default checkpoint date resolution now returns last NSE trading day on market-closed days.
- Checkpoint key TTL now targets `09:00 IST` of next actual NSE trading day (holiday-aware).
- Empty state wording improved for historical board dates (`Not captured`).

### Fixed
- Weekend empty timeline issue caused by defaulting to current non-trading date.
- Unfilled late-day slots persisting without backfill.
